Add file attachment (photo/video) to the Freeform Post generation dialog in client/src/pages/admin-marketing.tsx.

## Goal
When creating a Freeform Post, admin can optionally attach photos or videos.
- If files are attached → uploaded to Supabase Storage first, then URLs passed to Claude
- Claude uses the actual uploaded content context to write a more accurate caption
- FAL.AI image generation is skipped if a real photo is attached (use the real photo instead)
