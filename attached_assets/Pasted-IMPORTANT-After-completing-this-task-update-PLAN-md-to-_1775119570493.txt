Add a user-friendly credit exhaustion error handler for FAL.AI video generation (Veo models).

## Task 1: Update server/services/falService.ts

In the veo2 and veo3 generation blocks, wrap the fal.run() call with specific error detection:

For both veo2 and veo3 cases, catch errors and check if it's a balance/credit error:

try {
  const result = await fal.run("fal-ai/veo3", { ... }) as any;
  ...
} catch (error: any) {
  const errorBody = error?.body?.detail || error?.message || "";
  if (
    error?.status === 403 ||
    errorBody.includes("Exhausted balance") ||
    errorBody.includes("locked") ||
    errorBody.includes("balance")
  ) {
    throw new Error("CREDIT_EXHAUSTED:fal.ai");
  }
  throw error;
}

Do the same for kling, kling-3, veo2, veo3 cases — all video models.

Catch the CREDIT_EXHAUSTED error and return a specific response:

} catch (error: any) {

}

Update generateImageMut onError:
onError: async (err: Error) => {
  // Try to parse the error response
  try {
    const response = (err as any).response;
    if (response) {
      const data = await response.json();
      if (data.error === "CREDIT_EXHAUSTED") {
        setCreditAlert({
          open: true,
          service: data.service || "FAL.AI",
          chargeUrl: data.chargeUrl || "https://fal.ai/dashboard/billing",
        });
        return;
      }
    }
  } catch {}
  toast({
    title: "이미지/영상 생성 실패",
    description: err.message,
    variant: "destructive",
  });
},

Also update apiRequest call in generateImageMut to preserve the response:
mutationFn: async ({ id, model }: { id: string; model: string }) => {
  const response = await fetch(`/api/admin/marketing/queue/${id}/generate-image`, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    credentials: "include",
    body: JSON.stringify({ model }),
  });
  if (!response.ok) {
    const data = await response.json();
    const err = new Error(data.message || "Generation failed") as any;
    err.response = { json: () => Promise.resolve(data) };
    throw err;
  }
  return response.json();
},
